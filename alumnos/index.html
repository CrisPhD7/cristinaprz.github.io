Project Idea 1: 3D-Printed Finger-Clip Housing for Pulse Oximeter

Area: Biomedical Instrumentation & 3D Design

Available for 2026/2027

Description: The optical signal acquired by the DFRobot Fermion MAX30102 pulse oximetry sensor is highly sensitive to ambient light, which introduces noise and artifacts into physiological recordings. This project aims to design and manufacture a custom 3D-printed housing to encapsulate the sensor, effectively isolating it from external light sources to improve signal quality.

Key Design Requirements

  • Ergonomic Clip: A finger-clip mechanism for comfortable, reliable, and light-tight finger placement.
  • Modular Mounting: A mounting system that holds the sensor board firmly during use, but allows for easy tool-less removal to visually inspect the bare electronics or replace the board.

Roadmap & Implementation

  1. Software Selection: Evaluate and select appropriate 3D CAD software suitable for 3D printing.
  2. Dimensional Analysis: Precisely measure the sensor board, optical components, and connection pins.
  3. Prototyping: Design the clip mechanism, considering the mechanical properties required for repeated use.
  4. Material Selection & Budgeting: Select appropriate materials to block light and ensure comfort. Develop a comprehensive budget detailing the costs of all components, materials, and printing resources (required for university funding justification).
  5. Validation: Assemble the printed parts and perform signal acquisition tests to validate the reduction of ambient light interference.
